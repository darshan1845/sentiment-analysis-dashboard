# sentiment-analysis-dashboard
Real-Time Sentiment Analysis Dashboard Built a web-based NLP application using Python and Streamlit to analyze sentiment of user input and CSV files. Implemented real-time sentiment classification (Positive/Negative/Neutral), data visualization using Matplotlib, and interactive dashboard features for bulk and single text analysis.
